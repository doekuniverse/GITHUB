Comandos Intermedios

Ahora que dominas lo básico, explora procesos, usuarios y permisos.

Comando Descripción Ejemplo
ps Muestra procesos en ejecución ps aux
top Monitor de procesos en tiempo real top
kill Termina un proceso kill 1234
chmod Cambia permisos de archivo chmod 755 archivo
chown Cambia propietario chown usuario archivo
useradd Añade un usuario useradd nuevo_usuario
passwd Cambia contraseña passwd usuario
grep Busca texto en archivos grep "palabra" archivo
find Busca archivos find / -name archivo
tar Comprime/descomprime archivos tar -czf archivo.tar.gz dir

Consejos Avanzados

  • Usa kill -9 para forzar terminación.
  • Permisos: r=4, w=2, x=1. Total 755 para propietario full, grupo y otros read/execute.
  • Busca eficientemente: find . -type f -name "*.log" -mtime -7 archivos log modificados en 7 días.
