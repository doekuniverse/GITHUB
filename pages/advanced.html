Comandos Avanzados

Dominemos redes, seguridad, servicios y automatización en Amazon Linux.

Comando Descripción Ejemplo
systemctl Gestiona servicios systemd systemctl start httpd
yum Gestor de paquetes yum install paquete
firewall-cmd Configura firewall firewall-cmd --add-port=80/tcp
ss Muestra conexiones de red ss -tuln
curl Transfiere datos desde/hacia servidor curl http://example.com
wget Descarga archivos wget url
crontab Programa tareas crontab -e
journalctl Ve logs del sistema journalctl -u servicio
iptables Configura reglas de firewall iptables -A INPUT -p tcp --dport 22 -j ACCEPT
rsync Sincroniza archivos eficientemente rsync -av origen destino

Consejos Expertos

  • Usa systemctl enable para iniciar servicios al boot.
  • Yum: yum update mantiene el sistema seguro.
  • Automatiza con cron: 0 2 * * * backup.sh ejecuta backup diario a las 2 AM.
  • Monitorea logs: journalctl --since today
