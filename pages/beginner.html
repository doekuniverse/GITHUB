Comandos Básicos para Empezar

Estos comandos te ayudarán a navegar por el sistema de archivos y gestionar archivos básicos en Amazon Linux.

Comando Descripción Ejemplo
pwd Muestra el directorio actual pwd
ls Lista archivos y directorios ls -l (detallado)
cd Cambia de directorio cd /home
mkdir Crea un directorio mkdir nuevo_dir
touch Crea un archivo vacío touch archivo.txt
cp Copia archivos cp origen destino
mv Mueve o renombra archivos mv viejo nuevo
rm Elimina archivos rm archivo.txt
cat Muestra contenido de archivo cat archivo.txt
echo Imprime texto echo "Hola Mundo"

Consejos Creativos

  • Usa ls -la para ver archivos ocultos (empiezan con .)
  • Combina comandos: ls | grep txt filtra archivos .txt
  • Recuerda: ¡Siempre verifica antes de usar rm!
