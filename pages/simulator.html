💡 Atajos del Teclado:
• Tab - Autocompletado (próximamente)
• ↑ / ↓ - Navegar historial de comandos
• Ctrl+C - Cancelar comando actual
• Ctrl+L - Limpiar pantalla
• Escribe help para ver todos los comandos disponibles

💡 Consejos de Uso

  • Este simulador replica el comportamiento básico de una terminal Amazon Linux
  • Los comandos destructivos (rm, kill) están deshabilitados para seguridad
  • Los datos mostrados son simulados y no afectan ningún sistema real
  • Ideal para practicar sintaxis antes de usar comandos en una instancia EC2 real
  • Experimenta libremente - ¡no puedes romper nada!
